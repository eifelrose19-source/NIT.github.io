# NIT.github.io
Nations Integrated Therapy Website
This repository contains the source code for the official website of Nations Integrated Therapy, the practice of Dr. Viann Nations, Ph.D. The site serves as a digital gateway for clients in Idaho's Treasure Valley and Alaska to access mental health resources and counseling services.
 Project Overview
The website is designed to reflect the practice's mission of offering Hope, Healing, and Meaning through a holistic approach to mental health. It provides information on:
Dr. Viann Nations: Licensed Psychologist serving Idaho and Alaska.
Holistic Services: Focuses on the physical, intellectual, emotional, mindful, social, and spiritual aspects of well-being.
Counseling Options: Individual, Family, Marital, and Group counseling, including tele-mental health for Alaska.
 Built With
HTML5/CSS3: Used for structure and styling.
GitHub Pages: Used for hosting and deployment.
Site Structure
The website is organized into several key sections to ensure ease of navigation for potential clients:
Home: Introduction to the practice's philosophy and core values.
About Dr. Nations: Professional background, licensing information, and the symbolism of the practice's logo.
Services: Detailed breakdown of the holistic care model.
Contact: Information for reaching out to the practice.
